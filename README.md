# KernelSU build tester for non-GKI/GKI1 Kernel

## Introductions

- Since KernelSU v1.0.0, the support for non-GKI/GKI1 kernel has been deprecated.

- This repository's main goal is to test compilation status on old/legacy kernel, for easier older kernel compatibility debug while adding a new features or syncing from upstream.
